## Reading Width
- Article body text: `max-width: 65ch` (optimal for readability)
- Content sections: `max-width: 800px`

## Line Height
- Body: 1.7
- h1: 1.15, h2: 1.25, h3: 1.3, h4: 1.35, h5: 1.4, h6: 1.4
- Small/caption: 1.5

## Style Rules
- Headings: No margin-top, margin-bottom per table above
- Links: No underline by default, underline on hover
- All text uses CSS custom properties for theme switching
- Lists: margin-bottom 0.5em, nested lists indented 1.5em

## Font Weight Map
| Token | Usage |
|-------|-------|
| 400 (Regular) | Body text, small, captions |
| 500 (Medium) | h4, h5, h6, emphasized body text |
| 600 (Semi-bold) | h2, h3, section headers |
| 700 (Bold) | h1, hero headings, strong emphasis |

## Guidelines
- **Minimal text:** Keep headlines under 10 words where possible
- **Scannable:** Use short paragraphs, clear hierarchy
- **Direct language:** No consultant-speak, no corporate fluff
