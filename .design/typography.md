# Typography - No Excuse AS

## Font Family
- **Primary:** 'Roboto', sans-serif
- **Fallback:** System sans-serif

## Hierarchy
| Element | Desktop | Mobile (<=599px) |
|---------|---------|-----------------|
| h1 | 3.5em | 2em |
| h2 | 2em | 1.5em |
| h3 | 1.4em | 1.2em |
| Body | 1em (16px base) | 1em |

## Line Height
- Body: 1.7
- h1: 1.15, h2: 1.25, h3: 1.3

## Style Rules
- Headings: No margin
- Links: No underline by default, underline on hover
- All text uses CSS custom properties for theme switching

## Guidelines
- **Minimal text:** Keep headlines under 10 words where possible
- **Scannable:** Use short paragraphs, clear hierarchy
- **Direct language:** No consultant-speak, no corporate fluff
