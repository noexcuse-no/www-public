# JavaScript Patterns — No Excuse AS

## Approach

- Vanilla JS only — no frameworks (React, Vue, etc.)
- DOM-centric with inline event handlers
- No external dependencies

## Files

- `dark-mode-toggle.js` — toggles `dark-mode` class on body, controls dark stylesheet enabled state
- `profile-card.js` — toggles `.profile-expanded` / `.profile-compact` visibility, manages `active` / `inactive` classes
