# Page Layouts - No Excuse AS

## Homepage Layout

### Hero Section (Above Fold)
- Logo (left aligned)
- Tagline: Clear value proposition (<3 seconds to understand)
- Primary CTA: "Snakk med oss" / "Bestill samtale"
- Service highlights (icons)

### Services Section
- 2 service cards (Ledelse 60:2, Katalysator)
- Icon + title + brief description
- Direct CTA per service

### About Section
- Brief company positioning
- Target audience alignment

### Contact/Footer
- Contact information
- Microsoft Bookings integration
- Social links (LinkedIn)

## Service Pages (Ledelse 60:2, Katalysator)

### Structure
1. **Problem statement** - What problem do we solve?
2. **Solution approach** - How do we solve it?
3. **Proof/case examples** - Evidence
4. **Clear CTA** - Next action

## Profile Pages

### Structure
- Photo
- Name + title
- Tags/expertise
- Bio (brief)
- LinkedIn link
- Booking link

## Mobile-First Breakpoints
- **Mobile:** 320px - 599px
- **Tablet:** 600px - 1023px
- **Desktop:** 1024px+

## Touch Targets
- Minimum 44px height for all interactive elements
- Adequate spacing between tap targets

## Content Philosophy
- Minimal text - short, punchy headlines
- Visual-first - icons, graphics, clear hierarchy
- Direct CTAs - one action per section
- Scannable content structure
