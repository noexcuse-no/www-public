# CSS Conventions

Full guide: `.design/css-architecture.md`, `.design/colors.md`

Key: kebab-case class names, `var(--property)` everywhere, `--*-light`/`--*-dark` variable suffixes, no inline styles, no `@import`, no `!important`.
